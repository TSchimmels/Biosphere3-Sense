## Biosphere3 Bells and Whistles

This file lists the visual/experiential enhancements requested and their status.

Completed:
- 3D biosphere viewer with sensor nodes and basic depth projection.
- Animated alert beacon when warnings/offline sensors exist.
- Mission Control full-screen mode.
- Live signal waveforms for key sensor types.
- Holographic floor grid in the 3D view.
- Interactive mini-map with hover tooltips.
- Theme selector (dark ops, arctic, deep-sea, desert).
- Voice alerts for warnings/offline events.
- Demo mode to simulate spikes and failures.
- Onboarding walkthrough overlay.
- Exportable presentation snapshot (PNG).
- Live story panel summarizing system status.
- Sensor clustering visualization (grid-based).

Excluded by request:
- Gamified status badges.
- Ambient soundscape.
